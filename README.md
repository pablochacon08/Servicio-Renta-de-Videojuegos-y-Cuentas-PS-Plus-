# Servicio-Renta-de-Videojuegos-y-Cuentas-PS-Plus
Este proyecto va permitir administrar el servicio de rentas de videos y cuentas de 
PlayStation Plus al realizar consultas y modificaciones a los registros de clientes, rentas 
realizadas, empleados registrados, transacciones realizadas y multas aplicadas. Todo esto 
siguiendo la implementación CRUD (añadir/consultar/editar/eliminar registros).
